## 3. 外观设置

### 3.1 主题切换

```
[亮色] [暗色] [跟随系统]
```

| 状态 | 按钮 |
|------|------|
| 当前选中 | `variant="primary"` (accent bg) |
| 未选中 | `variant="outline"` (border) |

### 3.2 主题模式

| 模式 | 触发 | CSS |
|------|------|-----|
| 亮色 | `setMode("light")` | `.theme-light` |
| 暗色 | `setMode("dark")` | `.theme-dark` |
| 跟随系统 | `setMode("system")` | `prefers-color-scheme` media query |

```
   提供商: [OpenAI ▼]  ┌────────────┐
           │ OpenAI    │
           │ Anthropic │
           └────────────┘
   API Key: [••••••••••••••••]
```

### 4.3 模型选择

当前模型下拉框：

```
[ollama/llama3.2 ▼]
┌──────────────────────┐
│ ollama/llama3.2(120ms)│
│ ollama/mistral(85ms)  │
│ ollama/nomic(150ms)   │
│ openai/gpt-4o(离线)    │
└──────────────────────┘
```

| 状态 | 样式 | 禁用 |
|------|------|------|
| 可用 | 正常显示 (包含延迟) | false |
| 离线 | 模型名 + "(离线)" | true |

### 4.4 状态指示

```
● 在线 (status === "ready")
○ 无可用模型 (status === "no-model")  → yellow dot
○ 离线 (status === "no-model")        → red dot
```
