# Workspace 管理首页 — 交互原型 & 高保真

## 1. 流程概述

```
应用启动
    │
    ├── 已有最近 Workspace ──→ 自动打开上次 workspace
    │
    ├── 首次使用 (onboarded=false) ──→ OnboardingDialog 引导
    │       │
    │       └── 选择/创建 Workspace ──→ 进入主界面
    │
    └── Onboarding 已跳过 ──→ WelcomeView (空状态)
```

### 交互说明

| 元素 | 交互 |
|------|------|
| "选择本地文件夹"卡片 | 点击触发 Tauri 原生文件夹选择器 → openWorkspace |
| "导入示例笔记"卡片 | 点击复制示例文件到 workspace → 打开 editor |
| "开始使用" | markOnboarded() → 关闭 Dialog |
| "跳过" | markOnboarded() → 关闭 Dialog，显示 WelcomeView |

### 状态矩阵

| 状态 | 表现 | 触发条件 |
|------|------|----------|
| 空状态 (无 recent) | 不显示 "最近的知识库" 区块 | 首次安装 |
| 有 recent | 显示最近列表，点击直接打开 | 至少打开过一个 workspace |
| 加载中 | Card 按钮 disabled + text "加载中..." | 点击卡片后 |
| 错误 | prompt 内告知路径不存在 | 路径无效 |

## 4. Workspace 切换 (TopBar Dropdown)

```
  ┌──────────────────────────────────┐
  │  知识库: MemLab            ▼     │
  ├──────────────────────────────────┤
  │  📁 MemLab              ✓        │
  │  📁 dev-notes                    │
  │  📁 project-x                    │
  ├──────────────────────────────────┤
  │  打开知识库...                    │
  └──────────────────────────────────┘
```

### 交互说明

| 操作 | 行为 |
|------|------|
| 点击下拉 | 显示 recent workspaces 列表 |
| 点击已有 workspace | `openWorkspace(path)` → 刷新树 |
| 点击"打开知识库" | 弹出系统文件夹选择器 |
| 当前 workspace 标记 ✓ | 最近打开的被选中态 |
