# 知识图谱视图 — 交互原型 & 高保真

## 1. 布局

知识图谱作为 RightPanel 的一个 Tab (`mode="tree"`)：

```
┌─────────────────────────────────────────────────────────┐
│ [🔗] [☰] [🌐] [🤖]  ← TabBar (RightPanel tabs)  [×]  │
├─────────────────────────────────────────────────────────┤
│ 🔍 [筛选节点...________________]   (15)节点·(23)边      │
│                              [−] [100%] [+] [⊞]        │
├─────────────────────────────────────────────────────────┤
│                                                          │
│              ⬤─⬤                                        │
│            ╱     ╲                                       │
│          ⬤       ⬤──⬤                                   │
│          │       │   │                                   │
│          ⬤───────⬤   │                                   │
│                    ╲  │                                   │
│                      ⬤                                   │
│                                                          │
│  ┌───────────────┐                                       │
│  │ Agent API     │  ← Selected node detail card          │
│  │ 类型: note    │                                       │
│  │ [打开] [关闭] │                                       │
│  └───────────────┘                                       │
└─────────────────────────────────────────────────────────┘
```

## 2. 图谱交互

### 2.1 节点类型

| 节点类型 | 颜色 | 图标/形状 | 大小 |
|----------|------|-----------|------|
| note | accent(蓝) | circle | 6 + degree×1.5 |
| memory | success(绿) | circle | 6 + degree×1.5 |
| concept | warning(黄) | circle | 6 + degree×1.5 |
| agent | accent(紫) | circle | 6 + degree×1.5 |

### 2.2 边类型

| 边类型 | 样式 | 说明 |
|--------|------|------|
| reference | 实线 `---` | `[[双链]]` 引用 |
| tag | 虚线 `- - -` | 共享标签 |
| semantic | 点线 `····` | 语义相似度关联 |
| embed | 细线 `---`(0.5px) | 包含关系 |

### 2.3 交互

| 操作 | 行为 |
|------|------|
| 单击节点 | 选中高亮 (circle fill accent)，显示详情卡片 |
| 双击节点 | openFile(referenceId) |
| 单击空白 | 取消选中 |
| 拖拽 | (自动力导向布局，不可拖拽) |
| 滚轮缩放 | zoom +/- 10% (0.5x ~ 2.5x) |
| 筛选 | 输入过滤 label，graph 过滤重绘 |
| 悬停 | cursor: pointer，opacity highlight 邻接边 |

### 2.4 选中态的邻接高亮

```
选中 Node A:

  ┌─ A ─┐         ┌─────┐
  │ ⬤  │───⬤    │ ⬤  │ (opacity 1.0)
  └─────┘         └─────┘
    │               │
  ⬤               ⬤ (opacity 0.3)
  (opacity 1.0)   (不相关节点 fade)
```

未选中 | 选中时选中节点 | 选中时邻接节点 | 选中时非邻接节点
--|---|---|---
1.0 | 1.0 | 1.0 | 0.3

### 2.5 力导向布局参数

| 参数 | 值 |
|------|-----|
| 斥力常数 | 1800 |
| 连接距离 | 80px |
| 连接强度 | 0.08 |
| 阻尼 | 0.85 |
| 中心引力 | 0.005 |
| 最大迭代 | 300 帧 (~10s) |

## 4. 工具栏

```
┌──────────────────────────────────────────────────┐
│ 🔍 [filter input...]  (n 节点 · m 边)  [−] 100% [+] [⊞] │
└──────────────────────────────────────────────────┘
```

| 元素 | 交互 |
|------|------|
| 搜索输入 | 实时过滤节点 label，动态重绘 |
| 节点计数 | 显示当前 visible 节点和边的数量 |
| Zoom - / + | 缩放，步进 10% |
| Zoom 显示 | 当前百分比，居中 |
| Reset ⊞ | `setZoom(1)` 重置 |
