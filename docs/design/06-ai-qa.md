| 元素 | 表现 |

## 4. 当前精炼模式的状态矩阵

| 状态 | UI |
|------|-----|
| 未选中文本 | "选中编辑器内的文本，按 ⌘⇧E 使用 AI" |
| 已选中文本 | 显示 origin 区块和指令输入 |
| loading | "◌ AI 正在思考..." (animate-pulse) |
| 结果就绪 | 显示 refinded 区块 + 差异对比 |
| 错误 | "⚠ 错误信息" (danger 色框) |

## 5. Quick Actions

| 按钮 | 指令内容 |
|------|----------|
| ✨ 精炼 | "对该段落进行精炼，去冗余但保留事实" |
| 📋 摘要 | summarize API |
| 🎨 改写更专业 | "用更专业、正式的语气改写" |
| 🌐 翻译为英文 | "翻译为流畅英文，保持术语一致" |

## 7. 差异对比 (DiffView)

差异对比以可折叠 `<details>` 呈现：

```
▶ 差异对比 (collapsed)

▼ 差异对比 (expanded)
┌─────────────────────────────────┐
│ - 原始段落行1                    │
│ - 原始段落行2                    │
│ + 精炼后段落行1                  │
│ + 精炼后段落行2                  │
└─────────────────────────────────┘
```

| 行类型 | 背景色 | 文本样式 |
|--------|--------|----------|
| 删除行 (-) | `--color-diff-delete` | line-through, text-secondary |
| 新增行 (+) | `--color-diff-insert` | text-primary |
