- 采用 Tauri Command 模式暴露 API：`workspace.rs` 处理工作区创建/打开/配置，`file.rs` 处理底层文件读写与目录遍历，`editor.rs` 提供语言检测与 JSON 格式化。
- 数据持久化通过 Repository 模式实现：`WorkspaceRepo`、`NoteRepo` 等封装 `rusqlite` 连接，负责 SQL 执行与模型转换。
- 领域模型定义在 `models` 包下，使用 `serde` 进行序列化，区分视图模型（View）与请求模型（Request）。
- 依赖方向为 Commands -> Repositories -> Database (SQLite)，Commands 直接调用 `std::fs` 进行物理文件操作。
