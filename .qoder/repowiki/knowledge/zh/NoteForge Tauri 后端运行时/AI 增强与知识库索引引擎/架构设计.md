- **分层架构**：模块采用清晰的三层结构，`commands/` 层处理 Tauri IPC 请求，`ai.rs`、`knowledge.rs`、`vector.rs` 封装核心业务逻辑，`pipeline.rs` 协调多步原子化索引流程。
- **数据持久化**：深度依赖 `rusqlite`，利用 FTS5 虚拟表实现全文检索，通过 JSON 字段存储向量嵌入，并使用 `graph_nodes`/`graph_edges` 表维护知识图谱关系。
- **外部集成**：通过 `reqwest` 异步调用 Ollama API 进行 LLM 推理，使用 `fastembed` 在本地生成文本向量嵌入。
- **事务管理**：`IndexPipeline` 显式控制数据库事务（BEGIN/COMMIT/ROLLBACK），确保笔记元数据、索引、向量和图谱边在写入时的原子性。
