- 所有 Tauri 命令函数均接收序列化的 Request 结构体作为第一个参数，并通过 `State<T>` 获取数据库或存储实例。
- 数据模型结构体统一使用 `#[serde(rename_all = "camelCase")]` 以确保与前端 JavaScript 命名规范兼容。
- 数据库查询结果通过 `query_map` 配合闭包手动映射为领域模型，并使用 `filter_map(|r| r.ok())` 过滤潜在的行解析错误。
- 涉及多步写入的操作（如创建记忆并关联标签）在命令层通过显式调用多个 Repository 方法完成，而非在 Repository 内部隐式处理跨表逻辑。
