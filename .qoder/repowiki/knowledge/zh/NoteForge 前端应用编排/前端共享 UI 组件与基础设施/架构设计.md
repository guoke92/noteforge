- **UI 层**：基于 React 的无头/原子化组件（`src/components/ui/`），采用 `forwardRef` 与 `clsx/tailwind-merge` 实现样式组合。
- **状态层**：使用 Zustand 管理主题、编辑器会话、AI 配置等全局状态（`src/store/`），逻辑与视图分离。
- **通信层**：`src/ipc/` 封装统一的 IPC 桥接，通过检测 Tauri 环境自动切换真实后端调用或内存 Stub（`stub.ts`），支持离线开发与类型安全。
- **工具层**：`src/lib/` 包含 Monaco Editor 本地化配置、文件路径处理、Markdown 解析辅助及启动引导逻辑（`app-startup.ts`）。
- **Hook 层**：`src/hooks/` 封装键盘快捷键、文件拖拽等副作用逻辑，通过命令注册表与核心运行时交互。
