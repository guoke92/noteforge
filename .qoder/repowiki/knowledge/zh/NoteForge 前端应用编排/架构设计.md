- **启动编排**：在 `main.tsx` 中按序执行主题加载、核心服务初始化 (`initCore`)、生命周期安装及异步引导 (`startAppBootstrap`)，确保应用在 Splash 界面后进入就绪状态。
- **运行时单例**：`core/runtime.ts` 维护全局 `CoreRuntime` 单例，集中管理事件总线、文档、知识库、工作台等服务的依赖注入与生命周期。
- **跨模块桥接**：通过 `core/bridge` 目录下的同步逻辑（如 `editor-sync.ts`），将核心文档状态变更实时映射到编辑器 UI 标签页，实现数据层与表现层的解耦同步。
- **全局状态协调**：利用 Zustand Store (`store/`) 管理 UI 布局（侧边栏宽度、面板开关）与应用状态（启动进度、会话恢复），并通过 React Context/Hooks 在组件树中分发。
