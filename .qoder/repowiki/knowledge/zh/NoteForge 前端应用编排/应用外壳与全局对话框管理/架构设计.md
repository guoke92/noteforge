- **入口与编排**：`src/main.tsx` 执行主题初始化与核心运行时启动，渲染 `App.tsx` 作为根组件。
- **布局架构**：`App.tsx` 采用 Flexbox 实现经典的 IDE 三栏布局（侧边栏、编辑区、右侧面板），集成 `Resizer` 处理拖拽调整宽度，并通过 `DialogHost` 和各类独立 Dialog 组件在根层级渲染全局模态视图。
- **顶部导航**：`TopBar.tsx` 提供全局菜单（文件、视图、工具）、工作区切换下拉框及主题/面板快捷开关，深度集成 UI Store 控制全局状态。
- **对话框体系**：`DialogHost.tsx` 基于 `dialog-store` 动态分发系统级确认框（如关闭未保存提示、文件冲突解决）；功能型对话框（如 `GlobalSearchDialog`、`CommandPaletteDialog`）则直接挂载于 `App` 根部，通过 UI Store 的布尔标志位控制显隐。
- **启动引导**：`AppSplashScreen.tsx` 根据 `startupStore` 的状态展示分步加载进度，并在就绪后通过淡出动画移交控制权。
