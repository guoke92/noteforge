- 所有交互遵循“本地优先”原则，核心操作即时响应，网络仅用于 AI 调用且不阻塞主流程。
- 全局快捷键保持一致性，如 Cmd+N 新建、Cmd+S 保存、Cmd+P 搜索、Cmd+B 切换侧边栏。
- 色彩系统使用 CSS 变量（如 --color-accent, --color-bg-primary）并严格区分 Light/Dark 模式取值。
- 组件状态统一包含 Default、Hover、Active、Disabled 四种基本态，部分组件增加 Error 或 Loading 态。
- 布局架构采用经典的 IDE 风格：顶部 TopBar、左侧 Sidebar、中间 Editor Area、右侧 Right Panel、底部 StatusBar。
