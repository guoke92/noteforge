- **分层结构**：以 `01-design-system.md` 为基础层（色彩、字体、布局），`02-08` 系列文档为功能模块层（工作区、编辑器、搜索等）。
- **技术栈约定**：明确前端采用 React + TypeScript + Tailwind + Radix UI，编辑器内核为 Monaco Editor，状态管理使用 Zustand。
- **设计原则**：确立“本地优先”、“内容即核心”、“渐进式复杂”和“一致可预测”四大核心原则，指导所有界面与交互设计。
- **组件规范**：详细定义了 Button、Input、Card、Tree Node 等基础组件的状态（Default/Hover/Active/Disabled）及 Dialog、Dropdown 等复合组件的布局与行为。
